Version 2.

This mod prohibits most malicious actions such as, restraining players, construction,
place tents, barrels, digging garden plots, destroy and dismantle constructions, dismount
tents and much more. Also mod disables spawn zombies and animails in trade zones
and nearby.

HowTo add:

Server:
Put @TraderFix_v2 after @Trader in -mod start parameter of your server like this:
-mod=@Trader;@TraderFix_v2
Copy .bikey file from folder @TraderFix_v2\Key to folder Keys in root folder of your server

Client:
Subscribe to mod
Wait for mod downloads in Steam
Run DayZ Launcher (right mouse click on DayZ and choose Run...)
In Launcher:
Remove mod @Trader
Add mod @Trader
Add mod @TraderFix_v2
Mod @TraderFix_v2 should be in list of mods below @Trader (desirable!)

Enjoy!

Этот мод запрещает большинство вредоносных действий в торговых зонахю. Например, заковывание игроков, строительство,
установку палаток, бочек, вскапывание грядок, разрушение и разбор построек, демонтаж палаток
и многое другое. Так же мод отключает спавн зомби и животных в трейд-зонах и рядом с ними.

Как подключить:

Сервер:
Разместите мод @TraderFix_v2 после @Trader в параметре запуска сервера -mod как тут:
-mod=@Trader;@TraderFix_v2
Скопируйте файл .bikey из папки @TraderFix_v2\Key в папку Keys в корневой папке сервера

Клиент:
Подпишитесь на мод
Подождите когда он загрузится в стиме
Запустите DayZ Launcher (Щелкните правой кнопкой мыши на DayZ и выберете Запустить...)
В Launcher'е: 
Снимите выбор с мода @Trader
Добавте мод @Trader
Добавте мод @TraderFix_v2
Мод @TraderFix_v2 должен быть в списке ниже @Trader (желательно!)

Наслаждайтесь.
